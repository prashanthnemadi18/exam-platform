# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2025-07-15

### Added
- AI-powered face detection using TensorFlow.js
- Advanced cheating detection with face-api.js
- Smart proctoring component with behavior analysis
- Identity verification using face recognition
- Emotion analysis (7 emotions)
- Gaze tracking system
- Head pose analysis
- Suspicion scoring (0-100%)
- MongoDB database integration
- Mongoose models for Students, Exams, Submissions
- Groq AI support for ultra-fast inference
- Comprehensive documentation guides

### Changed
- Migrated from Vercel KV to MongoDB
- Updated database layer to use Mongoose
- Enhanced camera proctoring with AI
- Improved violation detection system

### Removed
- Old Vercel KV implementation
- Duplicate environment files
- Temporary build artifacts

### Security
- Face verification for exam integrity
- Multiple face detection
- Person switching detection
- Real-time monitoring
