## ⚠️ IMPORTANT: Add Environment Variable

Your website needs the Google AI API key to work properly.

### Add API Key:

1. Go to: https://vercel.com/prashanthnemadi-5995s-projects/online-platfrom/settings/environment-variables

2. Click **"Add New"**

3. Fill in:
   - **Name:** `GOOGLE_GENAI_API_KEY`
   - **Value:** Your Google AI API key
   - **Environment:** Check all (Production, Preview, Development)

4. Click **"Save"**

5. Go to **"Deployments"** tab and click **"Redeploy"**
